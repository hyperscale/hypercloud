Hypercloud a Cloud Application Platform based on Docker Swarm.

Install
-------

### Docker

```shell
docker pull hyperscale/hypercloud-server
```

### MacOS

Build hypercloud:
```shell
make build
```

Run hypercloud
```shell
make run-hypercloud-server
```
